Class: MoreDate

MoreDate()

new MoreDate()

A custom Date class that extends the built-in Date class with additional utility methods. This class provides static methods to parse dates, compare dates, and retrieve today's, tomorrow's, or yesterday's dates.

(static) parseDate(value) → {MoreDate|NaN}

Parses a value into a Date object. If the value is already a Date, it returns the Date object. If the value is a string, it attempts to convert it to a Date. If the value is of any other type or the string cannot be parsed, it returns NaN.
